# OSCEシステム テスト仕様書

## 1. 概要

### 1.1 ドキュメント情報
| 項目 | 内容 |
|------|------|
| システム名 | Digital Marksheet Exam System (OSCEシステム) |
| バージョン | 0.1.0 |
| 作成日 | 2026-04-26 |
| 対象環境 | Next.js 16 / React 19 / Supabase |

### 1.2 システム概要
大学医学部・歯学部のOSCE（客観的臨床能力試験）を運用するためのWebアプリケーション。
試験官（教員）・患者役・管理者の3ロールを持ち、試験部屋ごとの進行管理と評価入力、結果集計までをカバーする。

### 1.3 テスト対象機能
| # | 機能カテゴリ | 主要機能 |
|---|-------------|---------|
| 1 | 認証機能 | 管理者ログイン、教員ログイン、患者役ログイン |
| 2 | 試験管理 | テストセッション作成・管理、問題管理 |
| 3 | マスタ管理 | 大学管理、教科管理、部屋管理、ユーザー管理 |
| 4 | 試験実施 | 出席確認、評価入力、採点 |
| 5 | 結果管理 | ダッシュボード表示、結果集計、レポート |
