バージョンは [セマンティックバージョニング](https://semver.org/lang/ja/) に従います。

| 種別 | 例 | 意味 |
|---|---|---|
| **MAJOR** | 1.0.0 → **2**.0.0 | 運用の作法が変わる大きな仕様変更、データ構造の刷新 |
| **MINOR** | 1.0.0 → 1.**1**.0 | 新機能・新しい画面の追加(既存の使い方は変わらない) |
| **PATCH** | 1.0.0 → 1.0.**1** | バグ修正・表示の微調整 |

## [1.0.0] - 2026-07-21

**初回リリース。** 昭和医科大学 歯学部 OSCE(医療面接評価)の本番運用に向けた第 1 版です。

### 主な機能

**試験の準備**
- 試験課題・評価シートの作成／編集(シート → カテゴリー → 問題の階層)
- 段階数と配点をカテゴリー単位で設定し、問題ごとに個別上書きも可能
- 選択肢を数値だけでなく「はい / いいえ」などの文字でも設定可能
- アラート対象の選択肢を指定(該当時に管理画面へ警告表示)
- 大学 / 教科 / 部屋 / 教員 / 患者役 / 学生 のマスタ管理(CSV 一括取込対応)

**試験セッションの割当**
- 試験セッション単位で、学生・教員①②・患者役を部屋ごとに割当
- 統合 CSV による一括取込・エクスポート(CSV の教員①②の順番をそのまま保持)
- 1 名が複数部屋を担当する運用にも対応

**試験当日の評価入力**
- タブレットに最適化した評価入力画面(大きめの採点ボタン・選択解除対応)
- 教員①による出欠登録(出席／欠席／未確認のトグル)
- 教員①・教員②・患者役がそれぞれ独立して採点
- 大学／教科管理者による代理入力(どの役として入力するか選択可能)

**集計・確認**
- ダッシュボード:部屋別の進捗・完了状況・アラート・平均点を教員①②/患者役ごとに表示
- 受験者一覧:合計点・割合・合否判定・問題別の内訳、列名クリックでのソート、CSV 出力
- 合否判定は合格ラインを % で設定

**アカウント・権限**
- マスター管理者 / 大学管理者 / 教科管理者 / 一般教員 / 患者役 の権限体系
- 大学ごとのブランド設定(名称・アイコン)

### この版に含まれる直近の主な修正

- CSV で指定した教員①/教員②の順番が、画面上で入れ替わってしまう問題を修正
- 受験者一覧のデータが表示されない問題を修正
- 複数部屋を担当するアカウントが試験を選択できない問題を修正
- 受験者一覧の重複表示・並び順の修正(学籍番号順)、ステータス列を出欠表示に変更
- 画面デザインの全面刷新(状態色の整理・可読性向上・空状態の整備)
